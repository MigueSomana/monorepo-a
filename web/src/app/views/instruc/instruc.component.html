Instrucciones Básicas

Empleados

Crear Empleado: En este apartado puedes añadir un nuevo empleado con su Nombre y Correo

Editar Empleado: En este apartado puedes modificar la información actual del empleado (Nombre y Correo)

Información de Empleado: En este apartado puedes visualizar una tarjeta con toda la informacion del empleado

Departamentos

Crear Departamento: En este apartado puedes añadir un nuevo departamento con su Nombre y Descripción

Editar Departamento: En este apartado puedes modificar la información actual del departamento (Nombre y Descripción)

Información de Departamento: En este apartado puedes visualizar una tarjeta con toda la informacion del departamento, asi como ver su jerarquia de miembros

Eliminar Departamento: En este apartado puedes eliminar la información de un departamento de la lista

Añadir Miembro: En este apartado puedes añadir un miembro al departamento seleccionandolo por nombre y puedes asignarle un superior si hay disponibles

Ver Jeraquia de Miembros: Despliega el esquema de jerarquia de los miembros dentro del departamento

Para profundizar aun mas las intrucciones visualice el README dentro del repositorio :-)
